2.4 · Décisions structurantes

Quatre décisions ont posé le socle du module.

Section pivot de la restitution : ces choix structurent l'ensemble du modèle économique cible.

01

Les deux tracks fondamentaux Décision pivot

Distinction qui structure TOUS les calculs et TOUTES les vues du module.

Théoriquece qu'on pensait gagner
Réelce qu'on a réellement gagné/dépensé
Écartce qui explique la différence
 Théorique (Prévision / Conception)Réel (Réalisé / Exploitation)
DéfinitionPerformance attendue avant exécutionPerformance observée après exécution
CASomme prestations contrat (ADV)Somme prestations réellement effectuées
ChargesGrille de coûts + hypothèses paramétréesPaie (Silae/Factorial), carburant (AS24/Engie), péages, télématique
Indexation carburantOptionnel — vue hors indexation possibleIntégrée obligatoirement
02

Les unités d'analyse retenues

Six unités structurent la lecture du module à tous les niveaux.

UnitéRôleComplexité spécifique
ChauffeurCoûts RH (salaires, forfaits, amplitude)Alternance week-end → agréger sur 15 jours
TracteurCoûts détention, location, entretien, kilométrageDiviseur jours (21 vs 26) à décider
Semi-remorqueCoûts location, assurance, taux d'utilisationMutualisation entre tournées
TournéeMutualisation des ressources, mix clients/prestationsInclure pieds/repositionnements
Trajet / PrestationGranularité fine pour ventiler CA et chargesIndexation carburant, sous-traitance
Client / ContratRègles tarifaires, indexation, facturationFedEx = nombreuses lignes
03

Approche modèle « plage 24h optimisable »

« Concevoir une plage horaire 24h optimisable par tracteur, travailler la mutualisation (deux tournées sur un même véhicule) plutôt que de raisonner en SPL1/SPL2 isolés. »
04

Code couleur opérationnel

Vue conception : code couleur vert / orange / rouge pour les tournées, sans chiffres détaillés visibles côté planificateur, qui voit uniquement la santé économique globale de sa tournée.

2.7 · Risques majeurs verbalisés

Quatre risques identifiés en atelier.

🔴 Bloquant
Absence d'outil de gestion de parc centralisé
Contrats de location, ventilation coûts inter-entités, incohérences factures garage (ex. casse mêlée Avia Location / NAS Transport).
🔴 Bloquant
Décorrélation conception / exploitation
Planning non rétro-alimenté → CA non associable aux prestations réalisées sans import manuel.
⚠️ Levier majeur
600 k€ de coûts de non-utilisation véhicules
Sur l'exercice passé, non tracés dans l'outil actuel — gisement d'économie identifié.
🟠 Élevé
Identification du conducteur responsable d'une casse non fiable
Quand plusieurs conducteurs se sont succédé sur un même véhicule sur une période donnée.
